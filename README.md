## Инструкция запуска проекта
1. Клонировать репозиторий
```
git clone https://github.com/valerie-sidman/booking-service-front.git frontend
```
2. Перейти в директорию с проектом
```
cd frontend
```
3. Инициализировать проект
```
npm install
```
4. Запустить front
```
npm start
```
